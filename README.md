# ✨ Modern Todo List App

A sleek and interactive Todo List application built using React.js with a modern glassmorphism UI design.
This project allows users to add, complete, and remove tasks with smooth animations and a fully responsive interface.

## 🚀 Features

* ➕ Add new tasks
* ❌ Delete tasks instantly
* ✔️ Mark tasks as completed
* 🚫 Prevent duplicate tasks
* 🎨 Modern glassmorphism design
* 📱 Fully responsive layout
* ⚡ Smooth hover and transition effects
* 🌌 Animated vibrant background UI

## 🛠️ Tech Stack

* React.js
* JavaScript
* CSS3
* HTML5

## 📸 UI Highlights

* Full-screen modern layout
* Animated glowing background effects
* Interactive task cards
* Professional gradient theme
* Mobile-friendly design

## 📂 Installation

```bash
git clone <your-repo-link>
cd <repo-name>
npm install
npm start
```
